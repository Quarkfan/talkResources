# talkResources

分享主题内容仓库。包含所有对外分享主题的演示文稿、参考资料、思考记录和工具。

## 目录结构

```
├── presentations/    # 演示文稿 HTML
├── references/       # 参考资料（FAQ、防守弹药等）
├── thinking/         # 原始思考记录和讲者备注
├── themes/           # 主题主页（每个演讲主题一个）
└── tools/            # 工具项目（ai-tool-finder 等）
```

## 使用方式

1. 本地编辑内容
2. `git add . && git commit -m "说明"`
3. `git push`
4. talkshow 平台会通过挂载卷读取最新内容
