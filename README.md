# talkResources

分享主题内容仓库。包含所有对外分享主题的演示文稿、参考资料、思考记录和工具。

## 目录结构

```
├── presentations/    # 演示文稿 HTML（基于 HTML 的讲解资料）
├── references/       # 辅助参考资料（FAQ、防守弹药、外部素材等）
├── thinking/         # 原始资料（思路梳理、原始记录、讲稿草稿）
├── themes/           # 主题主页（每个演讲主题一个导航页）
└── tools/            # 辅助工具项目
```

## 内容组织规则

每个分享主题按以下目录分类存放：

- **`thinking/`** — 原始资料。收到内容后先放这里，包括思路碎片、原始笔记、讲稿草稿等。
- **`presentations/`** — 讲解资料。基于 HTML 制作的演示文稿，用于正式分享。
- **`references/`** — 辅助参考资料。用于分享时备查或应答的材料，如 FAQ、防守弹药、外部参考等。
- **`themes/`** — 每个演讲主题一个主题主页/导航页。
- **`tools/`** — 独立的辅助工具项目，每个工具一个子目录。

## 使用方式

1. 本地编辑内容
2. `git add . && git commit -m "说明"`
3. `git push`
4. talkshow 平台会通过挂载卷读取最新内容
