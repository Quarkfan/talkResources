## 使用提示

这些图片建议用于讨论以下主题：

- Bob 的“房间/住宅”界面隐喻
- Rover 作为拟人化助手的早期形态
- 从 Bob 到 Clippy，再到 Copilot / Agent 的交互演化
- 为什么 Bob 的设想在 1995 年失败，但在 AI Agent 时代可能以岗位型 agent 形式复活
